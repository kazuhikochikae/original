# README

## 開発言語

- Ruby 3.0.1
- Ruby on Rails 6.1.7

## 就業 Term の技術

- devise
- メッセージ機能

## カリキュラム外の技術

- active storage での画像表示

## 実行手順

```
git clone https://github.com/kazuhikochikae/original.git
cd original01
bundle install
rails db:create
rails db:migrate
rails s
```
